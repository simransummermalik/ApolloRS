# ApolloRS research manuscript

This directory is the manuscript workspace.  It currently contains a section
plan, not research results.  Values marked `TBD` must be generated from versioned
evidence; citations, benchmarks, coverage, and equivalence claims must not be
invented.

## Planned structure

1. Abstract — TBD after results
2. Introduction
3. Historical and technical background
4. Research questions and hypotheses
5. System architecture
6. Executable semantic model
7. Source recovery and translation methodology
8. Verification strategy
9. Experimental setup
10. Results — TBD
11. Divergence and negative-result analysis — TBD
12. Threats to validity
13. Limitations
14. Related work — citations TBD and source-checked
15. Future work
16. Conclusion — TBD after results

Tables and figures will be generated from raw artifacts by checked scripts.
Prose claims will link to the exact experiment, source/tool versions, fixture,
observable set, and evidence bundle.
